# Proctor-AI

📘 Proctor AI

AI-powered Online Exam Proctoring System

An end-to-end online exam platform with AI-based monitoring. It provides a student portal to take tests, an admin portal to create and manage tests, and a secure backend with real-time analytics. Built with ReactJS, Node.js, SQL, and TensorFlow.

⸻

🚀 Features
	•	🔒 Secure Login System – Authentication for students and admins
	•	🎓 Student Portal – Take tests, view results, and track performance
	•	🧑‍💼 Admin Portal – Create/manage tests and access student reports
	•	🤖 AI Monitoring – TensorFlow-based proctoring to flag suspicious activity
	•	📊 Result Analytics – Easy visualization of student performance
	•	📱 Responsive Design – Optimized for multiple screen sizes

⸻

🛠 Tech Stack

Layer	Technologies Used
Frontend	HTML, CSS, JavaScript, ReactJS
Backend	Node.js, Express.js
Database	SQL (MySQL/PostgreSQL)
AI/Monitoring	TensorFlow
Other Tools	Git, GitHub, Postman, VSCode


⸻

⚡ Getting Started

🔑 Prerequisites
	•	Node.js (v14+)
	•	npm or yarn
	•	SQL database (MySQL/PostgreSQL)
	•	TensorFlow installed

🔧 Installation

# Clone the repository
git clone https://github.com/AnushaReddy0711/Proctor-AI.git

# Go to project folder
cd Proctor-AI

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
npm install

▶ Run the Project

# Start backend
cd server
npm start

# Start frontend
cd client
npm start

The app will run at http://localhost:3000/.
